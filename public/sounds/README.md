# Sound Files for Calling System

This directory should contain the following audio files:

## Required Files

1. **ringback.mp3** - Played to the CALLER while waiting for the recipient to answer
   - Should be a standard phone-style ringback tone
   - Loops until call is answered or ends

2. **ringtone.mp3** - Played to the RECIPIENT when receiving an incoming call
   - Should be a notification-style ringtone
   - Loops until call is answered or declined

3. **message.mp3** - Played when a new message arrives (optional)
   - Should be a short, subtle notification sound
   - Single play (no loop)

## Where to Get Free Sound Files

You can download royalty-free sounds from:
- [Freesound.org](https://freesound.org/) - Search for "phone ring" or "ringtone"
- [Zapsplat.com](https://www.zapsplat.com/) - Free SFX library
- [Pixabay.com/sound-effects](https://pixabay.com/sound-effects/) - Royalty-free sounds
